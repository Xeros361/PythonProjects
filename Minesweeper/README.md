# Minesweeper – Console Python Game

A simple Minesweeper game that runs in the console or Windows CMD, written in Python.

---

## How to Play

1. Open your terminal or Command Prompt.

2. Go to the folder where the game file minesweeper.py is saved: cd path\to\folder

3. Run the game: minesweeper.py.

4. Enter the number of rows and columns for the board.

5. Choose a difficulty level: Easy, Medium, or Hard.

6. Play by entering row, column, and action:
	- c – click/uncover a cell

	- f – place or remove a flag
	
Example input:
Row, Column, Action: 2 3 c

7. The goal is to uncover all cells without mines.

---

## Requirements
- Python 3.x
- Runs in CMD, PowerShell, or any terminal
