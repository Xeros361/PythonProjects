# Minesweeper – Console Python Game

A simple Minesweeper game that runs in the console or Windows CMD, written in Python.

---

## How to Play

1. Download the file.

2. Open your terminal or Command Prompt.

3. Go to the folder where the game file minesweeper.py is saved: cd path\to\folder

4. Run the game: minesweeper.py.

5. Enter the number of rows and columns for the board.

6. Choose a difficulty level: Easy, Medium, or Hard.

7. Play by entering row, column, and action:
	- c – click/uncover a cell

	- f – place or remove a flag
	
Example input:
Row, Column, Action: 2 3 c

8. The goal is to uncover all cells without mines.

---

## Requirements
- Python 3.x

- Runs in CMD, PowerShell, or any terminal
